# タネマネー（Tane Money）— 現在の実装状態スナップショット
> 他AIへのアイデア出し用。2026年時点の現状サマリ。

## 1. プロダクト概要
- **何**：小学生〜中学生向けの**金融教育PWA**。お手伝い→ポイント→育成→投資体験で「お金の考える力」を育てる。
- **特徴**：実際のお金は動かず**すべてアプリ内ポイント**。「失敗しても1円も損しない」投資体験が売り。
- **期限性**：**中学卒業で卒業**（＝数年使う前提のプロダクト）。
- **本番**：https://tane-money.vercel.app
- **ビジネス目標**：高継続率（ほぼ100%）／年商¥10M規模。

## 2. 技術
- **単一ファイル** `okozukai-v9.jsx`（約10,800行）→ `index.html` にビルド。React/JSX、依存最小。
- **永続化**：Firestore（家族データ＝1つのJSON blob ＋ logsサブコレクション）＋ localStorage。匿名認証。
- **PWA**：build が version.json/sw.js を自動更新→「更新する」バーで全端末を最新に。
- **バックエンド**：`api/*`（Vercel Serverless）にStripe課金・FCMの土台あり（鍵未設定＝無効）。

## 3. モード／画面
- displayMode：`teen`(中高生)／`junior`(小学生)／`adult`(保護者)。
- タブ：毎日／活動／ためる／学ぶ／記録（junior は daily/tasks/goals）。

## 4. 現在の主な機能（純化後）
- **お手伝い/タスク**（good/bad/daily/回数）＋**承認制**（親が確認して反映）。
- **ポイント/お小遣い管理**、**目標(貯金)**、**ごほうび交換**。
- **タネモン育成**：卵→進化（分岐）、なでなで、図鑑、ガチャ、隠しモンスター/すがた、猫ライン、**ひみつ卵スラリル（今は卵のみ公開）**。
- **はたけ（投資シミュレーション・株のみ）**：
  - リアル株 約30銘柄（既定は**厳選13＋保有中**表示、「もっと見る」で全部）。
  - **架空の学習用銘柄5つ**（🌱コツコツ上げ/🚀乱高下/🎈急落/⛅横ばい/💎じわ上げ）。`fake`フラグでリアル取得せず**1日1回シミュ**。
  - **保護者が「上がりやすさ(bias)・ゆれ(vol)」を調整可**。
  - **畑メタファー**：株＝作物。買う＝種まき、保有日数で育つ、収穫＝売る。
- **使い道（自動家計簿）**：ごほうび交換ログから自動集計（手入力ナシ）。
- **学ぶ**：まめちしきを**順番のある8コース・カリキュラム**（9級→1級）＋クイズ。
- **ガチャ**：1日1回＋**バトルチケット**で追加（チケットはミッション/親付与から）。
- **ランキング**：競争→**応援化**（順位より「みんながんばってる」）。
- **成長レポート（親向け）・認定証・級**。
- **保護者の手綱**：投資ON/OFF、1日の売買回数上限、承認制、学習特化モード。
- **UX**：触覚フィードバック（Android／iOSは仕様で非対応）、**全ボタンの押し心地（スプリング＋リップル）**、**タネモンの励まし一言（やさしい/compassionate）**、reduced-motion対応。

## 5. 今回の純化で「外したもの」
- **モンスターバトル**：撤去（仕様は `docs/battle_system_spec.md` に保存、死んだコード約420行削除）。別アプリ化候補。
- **為替**：株に一本化（廃止）。
- **とっくんの旅（放置系冒険）**：入口撤去。
- **手入力家計簿**：自動「使い道」に置換。
- **回復薬・装備・バトル用HP報酬**：撤去。

→ アプリは **「育成 × 投資（株）× 金銭感覚 ＋ やさしいUX」** に純化。

## 9. 設計思想（ブレない軸）
- **誠実**：「儲かる話はしない」。投資は損もすると正直に。射幸性カット。
- **やさしい**：失敗を罰しない（Finch的）。達成は祝う（Duolingo的）。
- **純化**：機能を足すより引く。育成はタネモン、投資は株、と役割を分ける。
