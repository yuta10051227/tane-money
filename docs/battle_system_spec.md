> タネマネー本体からは撤去。**完全なソースコードはgit履歴に保存済み**（このコミットの1つ前まで `okozukai-v9.jsx` に存在）。
> 別アプリとして再構築する際の設計図。データ・式・コンポーネント構成をここに残す。

## 0. 全体像（コアループ）
タネモン（プレイヤー）が野生モンスターと**ターン制バトル** → 勝つと**かけら／装備ドロップ** → かけら5つで**ガチャチケット1枚** → 強い装備でボス「ヤミノオウ」討伐 → 稀に**ヤミノタマゴ**ドロップ（育てると究極体）。

## 1. データモデル（家族データ内）
| キー | 形 | 意味 |
|---|---|---|
| `monsterHP[cid]` | number | プレイヤー（タネモン）の現在HP。日替わり全回復＋1分1回復で持ち越し |
| `monsterHPDate[cid]` | "YYYY-M-D" | HPの当日判定 |
| `monsterHPTs[cid]` | epoch ms | 自然回復の起点 |
| `battleFragments[cid]` | number | バトルのかけら。5でチケット1枚 |
| `battleTickets[cid]` | number | ガチャもう1回チケット（※本体ガチャと共有。撤去後もミッションから供給） |
| `healPotions[cid]` / `healCaps[cid]{hs,hm}` | number | 回復アイテム所持 |
| `equip[cid]{weapon,shield}` | id | 装備中スロット（武器・盾） |
| `equipOwned[cid][]` | id[] | 取得済み装備 |
| `yamiEgg[cid]` | bool | ボス討伐で稀にドロップするヤミノタマゴ所持 |

## 2. 野生モンスター（WILD_MONSTERS）
`{name, emoji, lv, color, img, move:{n,e,c}}`。出現順＝lv順。
- スライムン🟢lv1 / コウモリン🦇lv2 / トゲちゃん🦔lv3 / ガイコツン💀lv4 / オニビ🔥lv5 / ヌシ・ドラゴ🐉lv7
- （高レベル帯）バクチン🎰lv22 / シャッキング⛓lv24 / ミエール🎭lv26 / ウマスギ🐍lv28 / ローヒー🌀lv30
  - ※高レベル帯は「お金の敵」モチーフ（ギャンブル・取り立て・見栄・うまい話・浪費）＝金融教育の寓意。
- **ボス** BOSS_MONSTER：ヤミノオウ👑lv11、技「ダークネスノヴァ🌑」、討伐で稀に `yamiEgg`。

## 3. 装備（EQUIPMENT）
`{id, slot:"weapon"|"shield", rarity, name, e, atk, def, hp, need:{k,v}, dropFrom, hint}`。
- 武器例：スライムソード(atk6)→…→ドラゴンクロー(atk22)→**ヤミノツルギ(atk28,hp14)**（ボスドロップ）。
- 盾例：きほんのたて→まもりのたて→…→**にじのオーラ(def13,hp42・目標5回達成で)**。
- 解放条件 `need.k`：`drop`（特定モンスター撃破ドロップ）/`tasks`（お手伝い累計）/`goals`（目標達成数・premium）。

## 4. 回復・サポート
- **HEAL_CAPS**：回復カプセル小(hs,+50)/中(hm,+100)。バトル勝利で低確率ドロップ。所持上限あり。
- **サポートなかま（SUPPORT）**：お手伝い数で加入（週3回ごと最大3体）。週替わりでタイプ固定。弱い子ほど相対的に効く補正。例：ラッキー🎲（攻撃/回復ランダム）。

## 5. 戦闘式（再現に必要な数値）
- 敵HP＝`50 + lv*28`。プレイヤー最大HP＝育成ステータス由来。
- プレイヤー技：`PLAYER_MOVES` から `pickMove(curId)`＝IDの文字コード和でmod選択（個体ごと固定）。
- ダメージ：atk・装備・サポート補正の合算（詳細はソース `BattleModal` 参照）。
- HP自然回復：**1分で1回復**（`regenHP(stored, ts, max)`）。日替わりで全回復。
- 報酬：勝利→かけら+1、`5たまったらチケット+1`（`dConv`）。装備/回復カプセルは低確率ドロップ。
- 演出：VS導入1.1s→`fight`フェーズ→こうげき→ヒット/プロジェクタイル→`result`。触覚 `buzz([30,60,30])`。

## 7. 別アプリ化のメモ
- このバトルは**単体で“RPG的お金教育ゲーム”**として独立可能（敵＝お金の誘惑、装備＝知識、ボス＝衝動）。
