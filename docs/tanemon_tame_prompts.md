# 🌱 タネモン「ためタネ」4表情 生成プロンプト（iPhone用・1枚＝1ブロック）
> まず1匹（ためタネ＝ためる/貯金タイプ）で画風を固める。OKなら のびタネ/つかいタネ/わけタネ に展開。
> 使い方：ブロックを長押し→コピー→画像生成アプリに貼るだけ。背景はチェッカー/マゼンタでOK（こちらで透過に抜いて128pxにします）。
> 送るときは「**ためタネ よろこぶ**」のように表情名を添えて。

## キャラ設定（4枚とも同じ姿に）
- 名前：**ためタネ**（ためる＝貯金タイプ）
- 姿：まるくてぷにっとした**タネの子**。抹茶グリーンのもち肌、頭に小さな芽、大きなキラキラの目、ほっぺにチーク、ちびの手足。
- 目印：おなかに**金色のコインの投入口（ブタ貯金箱）**＝「ためる」の証。
- 性格：コツコツ・しんぼう強い・やさしい。

## コツ
- **4枚とも「同じ姿・同じ色」**で、表情とポーズだけ変える → 並べたとき同一キャラに見える。
- 文字は入れない（看板の絵はOK、読める文字はNG）。
- まず「よろこぶ」1枚だけでもOK。かわいさを確認してから残り3枚→他の3匹へ。
